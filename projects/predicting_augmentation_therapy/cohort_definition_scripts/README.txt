This directory is used to go from the initial CPRD Depression cohort to an augmentation cohort.
In brief, this cohort
- Removes individuals who have never received an antidepressant
- Sets the index date to the first antidepressant date
- Removes individuals with pre-existing SMI
- Removes individuals with pre-existing epilepsy
- Removes individuals with any evidence of antipsychotic use which pre-exists antidepressant use.
